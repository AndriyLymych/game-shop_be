## Installation Docker

For running this project loccaly you should:
  - install Node.js v16.15.0
  - install and run Docker

## Installation project

```bash
$ npm install
```

## Installation db locally

```bash
$ npm run start docker:up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
